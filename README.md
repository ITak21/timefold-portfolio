# timefold-portfolio
> 해당 프로젝트는 java21, spring boot(3.5.5), h2로 개발하였습니다.


## 제작기간
>2025.09.15 ~ 2025.09.28 (14일)

## 제작의도
> 매 60분마다 들어오는 데이터를 하루치(1440분)으로 병합하여 유의미한 데이터를 만들고자 제작하였습니다.


## 주요기능

### 데이터 병합
- 60분 데이터를 설정된 기준에 맞춰 1440 데이터로 병합한다.
